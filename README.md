# Donnees Quebec (donneesquebec-ca)

Donnees Quebec is a provincial government open-data portal for Canada running **CKAN**, exposing a standardized catalog API over ~1,608 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/donneesquebec-ca/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=donneesquebec-ca-api-evangelist&utm_content=repo)

- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.3

## API
- **Donnees Quebec CKAN Action API** — base `https://www.donneesquebec.ca/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://www.donneesquebec.ca)
- Live endpoint verified: `https://www.donneesquebec.ca/api/3/action/status_show` · ~1,608 datasets
